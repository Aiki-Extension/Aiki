# Building the project ##
In order to build the project you need to complete the following steps.

## Step 1 ###

Install node.js (https://nodejs.org)

## Step 2 ###
Install packages:

```
npm install
```

## Step 3 ###
Build the project!

```
npm run build
```

To continuously rebuild upon file change:

```
npm start
```

To run all tests:

```
npm test
```
